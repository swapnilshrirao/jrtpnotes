###########
DB Tables
############


TABLE NAME :: ELIGIBILITY_DTLS

ELIG_ID				NUMBER		PRIMARY KEY
NAME				VARCHAR
MOBILE				NUMBAR
EMAIL				VARCHAR
GENDER				CHAR
SSN					NUMBER          (social security no in us   like PAN in india)
PLAN_NAME			VARCHAR
PLAN_STATUS		VARCHAR
PLAN_START_DATE	DATE
PLAN_END_DATE		DATE
CREATE_DATE		DATE
UPDATE_DATE		DATE
CREATED_BY		VARCHAR
UPDATED_BY 		VARCHAR


If we retriving the data then @GeneratedValue not needed..we just retrive the data from DB table
